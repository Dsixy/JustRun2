# Buff 与伤害

## 伤害类型

| 类型（代码键） | 中文名 | 典型来源 |
|----------------|--------|----------|
| `Physical` | 物理伤害 | 子弹、刀剑、爆炸、冲撞、敌人普攻等 |
| `Fire` | 火焰伤害 | 激光枪、灼烧 DoT |
| `Ice` | 冰霜伤害 | 冰霜手镯 |
| `Lightning` | 雷电伤害 | 激光剑、光轮切割附加雷伤 |
| `Poison` | 毒素伤害 | 毒药瓶、香水瓶、中毒 DoT |
| `Psychic` | 心灵伤害 | 心之歌、思维泡泡 |


## 元素反应

伤害类型作用于敌人**当前 Buff** 时的二级效果（`BuffElementReactions`）：

| 当前状态 | 中毒 | 冻伤 | 虚弱 | 灼烧 |
|---------|----|----|----|----|
| 火焰 | 中毒 tick 间隔 变为 0.4 秒 | 火焰伤害获得伤害加成(150+2.5等级+5启迪)%，同时清除冻伤 | 无 | 无 |
| 冰冻 | 持续时间延长至 7 秒 | 无 | 无 | 火焰伤害获得伤害加成(50+1等级+1.2启迪)%，清除一层灼烧 |
| 物理 | 无 | 物理伤害基础伤害增加(2+1等级+0.4启迪)点 | 无 | 无 |
| 雷电 | 中毒 tick 伤害增加 50% | 施加的闪电链范围增加至450，最大目标增加至6个 | 立刻造成 20 + 3启迪的心灵伤害，解除虚弱 | 扩散灼烧buff至闪电链目标，层数一致 |
| 毒素 | 无 | 无 | 无 | 无 |
| 心灵 | 无 | 无 | 无 | 无 |

> 闪电链衍生伤害不再次触发反应；反应衍生心灵伤害同理。

## Buff 

### 灼烧

- `ID`: 0
- `类名`: BurningBuff
- `来源`: 火箭发射器

可叠加 3 层，每秒对持有者造成 `2+层数×(1启迪+0.8等级)`**火焰伤害**，持续 4 秒。

持有**灼烧**时，会对收到的 `冰冻`，`雷电`伤害做出响应，效果见（# CITE）

### 中毒

- `ID`: 1
- `类名`: PoisonBuff
- `来源`: 毒药瓶，香水瓶

每 0.6 秒对持有者造成 `2+(0.4等级+5)×(0.3启迪+2)`**毒素伤害**，持续 4.5 秒

中毒效果会受到以下效果的改变：
- `火焰` `冰冻` `雷电`伤害会改变毒素的效果（#CITE）
- `香水瓶` 施加的中毒效果会受到 **香氛值** 的加成


### 冻伤
- `ID`: 2
- `类名`: FrostBiteBuff
- `来源`: 冰霜手镯

持有者速度降低 50 点。持续 5 秒。

持有 **冻伤** 时，会对收到的 `火焰`，`雷电`，`物理`伤害做出响应，效果见（# CITE）

### 无敌

- `ID`: 3
- `类名`: InvinceibleBuff
- `来源`: 雨滴茉莉

获得无敌效果，持续 10 秒

### 狂暴

- `ID`: 4
- `类名`: HasteBuff
- `来源`: 风信子

获得狂暴效果：攻击间隔减少 50%。持续 15 秒

### 猫咪护盾

- `ID`: 5
- `类名`: CatShieldBuff
- `来源`: 猫的计谋

获得一次免伤效果

### 超级 Pro

- `ID`: 6
- `类名`: SuperProBuff
- `来源`: 猫的计谋

使持有者单次收到伤害不会超过其最大生命值的 10%，持续 5 秒。

### 闪电

- `ID`: 7
- `类名`: LightningBuff
- `来源`: 光轮切割，激光剑

对持有者周围 150 范围内随机 3 个单位释放闪电链，造成 `4+0.6启迪+0.25等级` 的**雷电伤害**

### 虚弱

- `ID`: 8
- `类名`: WeakenBuff
- `来源`: 心之歌，思维泡泡，唤灵海螺

使持有者基础伤害降低 30%，持续 5 秒。

持有 **虚弱** 时，会对收到的`雷电`伤害做出响应，效果见（# CITE）

### 狂热

- `ID`: 9
- `类名`: FrenzyBuff
- `来源`: 镰刀先生

获取 buff时，立刻回复50%生命值，并在持续期间每秒恢复 15% 生命值，同时造成的伤害增加 50%，持续4秒。
