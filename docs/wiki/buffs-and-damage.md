# Buff 与伤害

## 伤害类型

| 伤害类型 | 典型来源 |
|----------|----------|
| 物理伤害 | 子弹、刀剑、爆炸、火箭炮直击、光轮主体伤害、冲撞、敌人普攻等 |
| 火焰伤害 | 激光枪、灼烧持续伤害 |
| 冰霜伤害 | 冰霜手镯 |
| 雷电伤害 | 激光剑、极速光轮（4 级光刃模式）、闪电链 |
| 毒素伤害 | 毒药瓶、香水瓶、中毒持续伤害 |
| 心灵伤害 | 心之歌、思维泡泡、唤灵海螺（灵体冲击） |

---

## 元素反应

当某种**伤害类型**命中敌人时，若敌人身上已有对应 **Buff**，会产生额外效果。公式中的「等级」「启迪」取自**造成该次伤害的来源**（武器或角色属性）。

| 命中伤害 ↓ ／ 敌人状态 → | 中毒 | 冻伤 | 虚弱 | 灼烧 |
|--------------------------|------|------|------|------|
| 火焰 | 中毒 tick 间隔变为 0.4 秒 | 火焰伤害获得加成 (150+2.5×等级+5×启迪)%，同时清除冻伤 | 无 | 无 |
| 冰冻 | 中毒持续时间延长至 7 秒 | 无 | 无 | 火焰伤害获得加成 (50+1×等级+1.2×启迪)%，灼烧层数 −1 |
| 物理 | 无 | 物理伤害基础值 + (2+1×等级+0.4×启迪) 点 | 无 | 无 |
| 雷电 | 中毒 tick 伤害 +50% | 闪电链范围增至 450，最大目标增至 6 个 | 立刻造成 20+3×启迪 心灵伤害，并解除虚弱 | 将灼烧层数扩散至闪电链目标 |
| 毒素 | 无 | 无 | 无 | 无 |
| 心灵 | 无 | 无 | 无 | 无 |

> 闪电链衍生伤害不再触发元素反应；由反应产生的额外心灵伤害同理。

> 灼烧、中毒等持续伤害在每次 tick 时也会结算受击，因此可能再次参与元素反应。

## 敌人状态（Debuff）

### 灼烧

- **来源**：火箭炮（4 级，爆炸附带）

可叠加 3 层。每秒对持有者造成 `2+层数×(1×启迪+0.8×等级)` **火焰伤害**，持续 4 秒。

持有者处于灼烧时，受到**冰冻**、**雷电**伤害会触发 [元素反应](#元素反应) 中的对应效果。

### 中毒

- **来源**：毒药瓶、香水瓶

每 0.6 秒对持有者造成 `2+(0.4×等级+5)×(0.3×启迪+2)` **毒素伤害**，持续 4.5 秒。

中毒效果会受到以下效果的改变：

- **火焰**、**冰冻**、**雷电**伤害会改变毒素的效果（见 [元素反应](#元素反应)）
- **香水瓶** 施加的中毒效果会受到 **香氛值** 的加成

### 冻伤

- **来源**：冰霜手镯

持有者移动速度降低 50 点，持续 5 秒。

持有者处于冻伤时，受到**火焰**、**雷电**、**物理**伤害会触发 [元素反应](#元素反应) 中的对应效果。

### 闪电链

- **来源**：激光剑（雷电伤害命中时）；极速光轮（4 级光刃模式的雷电伤害命中时）

命中时立即对持有者周围 **150** 范围内随机 **3** 个敌人释放闪电链，造成 `4+0.6×启迪+0.25×等级` **雷电伤害**；之后 **每 1 秒** 再释放一次，持续 **4 秒**。

若持有者带有冻伤，闪电链的范围与目标数可能因 [元素反应](#元素反应) 而提升；若带有灼烧，链击目标会同步获得相同层数的灼烧。

### 虚弱

- **来源**：心之歌、思维泡泡、唤灵海螺——当这些武器的 **心灵伤害暴击** 命中敌人时施加

使敌人基础攻击伤害降低 30%，持续 5 秒。

持有者处于虚弱时，受到**雷电**伤害会触发 [元素反应](#元素反应) 中的对应效果。

### 狂热 {#狂热}

- **来源**：镰刀先生技能 **威慑**（对场上所有敌人施加）

获得时立刻回复 **50%** 最大生命值；持续 **4 秒** 期间，每秒再恢复 **15%** 最大生命值，且造成的伤害 **+50%**。

> 与玩家拾取风信子获得的「狂暴」（攻速提升）不同，**狂热** 是施加在敌人身上的强化状态。

## 玩家增益（Buff）

### 无敌 {#无敌}

- **来源**：拾取雨滴茉莉

获得无敌，持续 10 秒。

### 狂暴 {#狂暴}

- **来源**：拾取风信子

攻击间隔缩短 50%（攻速提升），持续 15 秒。

### 猫咪护盾

- **来源**：猫之计谋——累计攻击达到要求次数后获得

抵挡下一次受到的伤害（可叠加层数，2 级起最多 2 层）。

### 超级 Pro

- **来源**：猫之计谋（4 级）——每次攻击时获得

单次受到的伤害不超过最大生命值的 10%，持续 **10 秒**。
