# Buff 与伤害

## 伤害类 `DamageInfo`

| 字段 | 说明 |
|------|------|
| `baseAmount` | 基础伤害 |
| `damageBonus` | 加成倍率 |
| `damageType` | 元素 / 物理类型（见下） |
| `isCrit` | 是否暴击 |
| **最终伤害** | `int(baseAmount × damageBonus)`（再经护甲、Buff 修正） |

## 伤害类型

| 类型 | 设计效果 | 实现 |
|------|----------|------|
| **贯穿** | 子弹、刺击 | ✅ |
| **劈砍** | 刀剑 | ✅ |
| **力场** | 爆炸、钝击 | ✅ |
| **火焰** | 可灼烧 | 部分武器 ⚠️ |
| **冰元素** | 施加冻伤 | 部分效果 ✅ |
| **电元素** | 闪电链等 | 📋 不完整 |
| **毒素** | 依赖中毒 Buff | ✅ |
| **心灵** | 暴击 → 虚弱 | 📋 虚弱未实现 |

## 已实现 Buff

| ID | 名称 | 类名 | 效果摘要 |
|----|------|------|----------|
| 0 | 灼烧 | `BurningBuff` | 可叠 3 层；4s；tick 1s；伤害 `2+层数×(0.4启迪+0.2等级)` |
| 1 | 中毒 | `PoisonBuff` | 4.5s；tick 0.6s；二级反应 📋 |
| 2 | 冻伤 | `FrostBiteBuff` | 减速；额外受伤 ⚠️ 数值与文档有差 |
| 3 | 无敌 | `InvincibleBuff` | 免伤 |
| 4 | 狂暴 | `HasteBuff` | 攻击间隔 ×0.5（玩家）；**威慑**（小怪 +50 伤）📋 |
| 5 | 猫咪护盾 | `CatShieldBuff` | 免疫一次伤害 |
| 6 | 超级 Pro | `SuperProBuff` | 单次受伤 ≤ 10% 最大生命 |
