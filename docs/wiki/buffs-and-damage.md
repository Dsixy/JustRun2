# Buff 与伤害

## 伤害类 `DamageInfo`

| 字段 | 说明 |
|------|------|
| `baseAmount` | 基础伤害 |
| `bonus` | 加成倍率 |
| `damageType` | 伤害类型（见下；代码常量 `DamageTypes`） |
| `isCrit` | 是否暴击 |
| **最终伤害** | `int(baseAmount × (1 + bonus) × critDamage)`（再经护甲、Buff 修正） |

## 伤害类型

> **物理子类已合并**：设计文档中的贯穿、劈砍、力场、钝击等，在代码与 wiki 中统一为 **物理伤害**（`Physical`）。  
> 贯穿次数、劈砍范围等仍作为**武器机制**描述，不再单独作为 `damageType`。

| 类型（代码键） | 中文名 | 典型来源 | 实现 |
|----------------|--------|----------|------|
| **`Physical`** | **物理伤害** | 子弹、刀剑、爆炸、冲撞、敌人普攻等 | ✅ 默认类型 |
| `Fire` | 火焰伤害 | 激光枪、灼烧 DoT | ✅ |
| `Ice` | 冰霜伤害 | 冰霜手镯 | ✅ |
| `Lightning` | 雷电伤害 | 激光剑、光轮切割附加雷伤 | ✅ |
| `Poison` | 毒素伤害 | 毒药瓶、香水瓶、中毒 DoT | ✅ |
| `Psychic` | 心灵伤害 | 心之歌、思维泡泡 | ✅ |

### 已废弃（映射为 Physical）

| 旧称 | 说明 |
|------|------|
| 贯穿 | 现为物理伤害；狙击「暴击贯穿」为 pierce 机制 |
| 劈砍 | 现为物理伤害 |
| 力场 / 钝击 | 现为物理伤害 |

## 已实现 Buff

| ID | 名称 | 类名 | 效果摘要 |
|----|------|------|----------|
| 0 | 灼烧 | `BurningBuff` | 可叠 3 层；4s；tick 1s；伤害 `2+层数×(0.4启迪+0.2等级)`；**火焰伤害** |
| 1 | 中毒 | `PoisonBuff` | 4.5s；tick 0.6s；**毒素伤害** |
| 2 | 冻伤 | `FrostBiteBuff` | 减速；额外受伤 ⚠️ 数值与文档有差 |
| 3 | 无敌 | `InvincibleBuff` | 免伤 |
| 4 | 狂暴 | `HasteBuff` | 攻击间隔 ×0.5（玩家）；**威慑**（小怪 +50 伤）📋 |
| 5 | 猫咪护盾 | `CatShieldBuff` | immune 一次伤害 |
| 6 | 超级 Pro | `SuperProBuff` | 单次受伤 ≤ 10% 最大生命 |
| 7 | 闪电 | `LightningBuff` | 周期向 150 内至多 3 名敌人释放闪电链；**雷电伤害** |
