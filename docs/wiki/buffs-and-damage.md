# Buff 与伤害

## 伤害类型

| 类型（代码键） | 中文名 | 典型来源 | 实现 |
|----------------|--------|----------|------|
| `Physical` | 物理伤害 | 子弹、刀剑、爆炸、冲撞、敌人普攻等 | ✅ 默认类型 |
| `Fire` | 火焰伤害 | 激光枪、灼烧 DoT | ✅ |
| `Ice` | 冰霜伤害 | 冰霜手镯 | ✅ |
| `Lightning` | 雷电伤害 | 激光剑、光轮切割附加雷伤 | ✅ |
| `Poison` | 毒素伤害 | 毒药瓶、香水瓶、中毒 DoT | ✅ |
| `Psychic` | 心灵伤害 | 心之歌、思维泡泡 | ✅ |


## 元素反应

伤害类型作用于敌人**当前 Buff** 时的二级效果（`BuffElementReactions`）：

| 当前状态 | 中毒 | 冻伤 | 虚弱 | 灼烧 |
|---------|----|----|----|----|
| 火焰 | 中毒 tick 间隔 变为 0.4 秒 | 火焰伤害获得伤害加成(150+2.5等级+5启迪)%，同时清除冻伤 | 无 | 无 |
| 冰冻 | 持续时间延长至 7 秒 | 无 | 无 | 火焰伤害获得伤害加成(50+1等级+1.2启迪)%，清除一层灼烧 |
| 物理 | 无 | 物理伤害基础伤害增加(2+1等级+0.4启迪)点 | 无 | 无 |
| 雷电 | 中毒 tick 伤害增加 50% | 施加的闪电链范围增加至450，最大目标增加至6个 | 立刻造成 20 + 3启迪的心灵伤害，解除虚弱 | 扩散灼烧buff至闪电链目标，层数一致 |
| 毒素 | 无 | 无 | 无 | 无 |
| 心灵 | 无 | 无 | 无 | 无 |

> 闪电链衍生伤害不再次触发反应；反应衍生心灵伤害同理。

## 已实现 Buff

| ID | 名称 | 类名 | 效果摘要 |
|----|------|------|----------|
| 0 | 灼烧 | `BurningBuff` | 可叠 3 层；4s；tick 1s；伤害 `2+层数×(1启迪+0.8等级)`；**火焰伤害** |
| 1 | 中毒 | `PoisonBuff` | 4.5s；tick 0.6s；**毒素伤害** |
| 2 | 冻伤 | `FrostBiteBuff` | 减速；额外受伤 ⚠️ 数值与文档有差 |
| 3 | 无敌 | `InvincibleBuff` | 免伤 |
| 4 | 狂暴 | `HasteBuff` | 攻击间隔 ×0.5（玩家）；**威慑**（小怪 +50 伤）📋 |
| 5 | 猫咪护盾 | `CatShieldBuff` | immune 一次伤害 |
| 6 | 超级 Pro | `SuperProBuff` | 单次受伤 ≤ 10% 最大生命 |
| 7 | 闪电 | `LightningBuff` | 周期向 150 内至多 3 名敌人释放闪电链；**雷电伤害** |
| 8 | 虚弱 | `WeakenBuff` | 5s；敌人基础伤害 ×0.7；**心灵暴击**触发 |

> 元素二级反应见上文 **元素反应** 表；旧版「火/冰/电 Buff ID 判定」已移除。

## 伤害飘字

击中敌人显示数字，颜色由 `DamageTypes.color()` 决定：

| 类型 | 飘字颜色 |
|------|----------|
| 物理 | 黄 |
| 火焰 | 红 |
| 冰霜 | 蓝 |
| 雷电 | 浅蓝 |
| 毒素 | 绿 |
| 心灵 | 粉 |

### 心灵 · 暴击

心灵伤害暴击命中时附加 **虚弱**（ID 8）：敌人基础伤害降低 30%，持续 5s；重复触发刷新持续时间。

升级显示「升级！」「天赋点+2」飘字（共用飘字组件）。
