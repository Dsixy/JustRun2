# 剧情与事件

事件由 `Events` 信号总线分发，结果写入 `GameInfo` 配置。

## 现网事件（10 波版）

| 事件键 | 触发 | 演出 | 持久化 |
|--------|------|------|--------|
| `first_to_wave_3` | 首次完成 **第 3 波** | 卡莫入场对话 → 解锁 **卡莫** + 赠送 **激光枪** | ✅ |
| `first_to_wave_6` | 首次完成 **第 6 波** | 阿女入场 → 解锁 **阿女** + **花刀** + **10 天赋点** | ✅ |
| `first_{属性}_to_20` | 通关结算时属性 = 20 | 解锁对应武器 | 部分 ✅ |
| `seen_popocat_hint` | 首局 | 商店操作提示 | ✅ |

旧存档键 `first_to_wave_10` / `first_to_wave_20` / `first_to_wave_7` 会自动迁移至 `first_to_wave_3` / `first_to_wave_6`。
