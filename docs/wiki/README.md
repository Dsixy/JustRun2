## 快速参考

- **单局长度**：10 波，波间波波猫商店
- **初始角色**：仅阿猴；第 3 波解锁卡莫、第 6 波解锁阿女（剧情后可选）
- **快捷键**：`B` 背包/武器 · `M` 天赋 · `T` 图鉴 · `Esc` 关闭 UI
- **升级**：+2 天赋点 · +3 商店刷新次数（可存储）
