# 敌人

所有普通怪有 **3 个等级**（外观深浅区分）。属性随 **等级** 与 **当前波次系数** 缩放。

## 波次系数

与 [波次](progression.md) 表一致：1, 2, 4, 8, 12, 24, 30, 36, 44, 50。

## 刷怪机制（代码）

- 屏幕外随机边生成  
- 间隔与 **场上敌人数量** 正相关  
- 击杀率反馈调节下一批 **数量倍率** `generateBonus`  
- 各波具体编队见 `scene.gd` → `enemyGenerate` 表  

与设计文档早期「0～5 波每次 3 豆豆、间隔 1+0.2×怪数」简化描述不同，**以分组表为准**。
