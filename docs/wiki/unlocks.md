# 解锁条件

元进度写入 `user://config.cfg`，分 **WeaponUnlock**、**CharacterUnlock**、**Event** 与 **Meta** 四节。

> **Meta**：跨局累计统计（植物拾取、按角色击杀等），由 `RunStats` 在局末写入。

## 默认解锁

| 类型 | 条件 |
|------|------|
| 手枪 | 新档默认 ✅ |
| 角色 **阿猴** | 新档默认 ✅（`CharacterUnlock.aho`） |

## 角色绑定武器（非全局解锁）

以下武器随角色场景自带，**不**写入 `WeaponUnlock`，也不进商店随机池的全局默认：

| 武器 | 绑定角色 |
|------|----------|
| 狙击步枪 | 阿猴 |
| 霰弹枪 | 卡莫 |
| 激光剑 | 卡莫 |
| 毒瓶 | 阿女 |

全局解锁仅影响商店池与图鉴；开局武装仍由所选角色决定（patch：不自选武器）。

## 累计 / 单局条件（RunStats）

| 武器 | 条件 | Meta / 单局字段 | 代码 |
|------|------|-----------------|------|
| 外卖员 | 单局消费 **4000** 金币 | `money_spent` | ✅ |
| 群星之怒 | 使用 **卡莫** 累计击杀 **5000** | `Meta.kills_carmor_total` | ✅ |
| 金币枪 | 单局获得 **10000** 金币 | `money_earned` | ✅ |
| 蛛丝 | 通关时武装 **均为 DoT** | 局末检测 `weaponArm` | ✅ |
| 香水瓶 | 累计拾取 **100** 植物 | `Meta.plants_picked_total` | ✅ |

**DoT 判定**：毒瓶、香水瓶、冰霜手镯；满级火箭炮（燃烧）视为 DoT。空槽位不参与检测，已装备的非 DoT 武器会导致条件失败。

## 战斗统计（RunStats Autoload）

单局记录：击杀、金币收支、植物拾取、波次、角色、等级、胜负。局末 `end_run()` 写入 Meta 并检测上表解锁；结算面板展示摘要与本局新解锁武器。

## 图鉴

- 获得过的武器可在 `T` **图鉴** 浏览  
- **不可** 用于开局自选（补丁：与角色绑定）
