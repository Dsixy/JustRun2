# 掉落物

## 经验宝石

获取经验的主要来源。击杀任意怪物，都有1.2%概率掉落经验宝石。经验宝石等级随波次改变。

| 等级 | 经验值 | 外观 |
|------|--------|------|
| 基础 | 10 | <img src=""> |
| 中级 | 50 | <img src=""> |
| 高级 | 500 | <img src=""> |

经验宝石等级概率如下：
| 波次 | 掉落 |
|------|------|
| 1 | 30% 基础 |
| 2 | 30% 基础，5% 中级 |
| 3 | 45% 基础，7% 中级 |
| 4 | 30% 中级 |
| 5 | 50% 基础，10% 中级 |
| 6 | 50% 中级，2% 高级 |
| 7 | 30% 基础，25% 中级 |
| 8 | 30% 高级 |
| 9 | 10% 基础，40% 中级 |
| 10 | 50% 高级 |

## 波次结束自动拾取

波次计时结束时，会依概率拾取全场的金币与经验宝石，未被拾取的将被销毁。

- 金币 **30%** 概率吸附  
- 经验宝石 **10%** 概率吸附
