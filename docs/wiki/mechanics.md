# 游戏机制

**Just Run2** 是俯视角 Roguelike 生存游戏：固定 **10 个波次**，波间在波波猫商店整备，用金币与天赋构筑武器，第 10 波击败 **镰刀先生** 通关。角色与装甲臂见 [角色](characters.md)；武器与解锁见 [武器索引](weapons/index.md)。

---

## 单局循环

```
选角（初始仅阿猴）
    ↓
波波猫商店
    ↓
波次战斗（计时生存 + 刷怪）
    ↓
拾取掉落 · 升级 · 拖拽武装
    ↓
（重复至第 10 波）
    ↓
通关或失败 → 主菜单
```

---

## 操作与界面

| 按键 | 功能 |
|------|------|
| `B` | 背包与武装板（7 格背包，4 槽武装板） |
| `M` | 天赋面板（分配八维属性） |
| `T` | 武器图鉴 |
| `Esc` | 关闭当前 UI |

将背包中的武器拖入上方武装板槽位后，由角色绑定的装甲臂按攻击段轮射。攻击段配置见各 [角色](characters.md) 条目。

## 波次 {#波次}

| 项目 | 说明 |
|------|------|
| 总波数 | **10** |
| 每波时长 | 30 / 35 / 40 / 45 / 45 / 50 / 50 / 50 / 55 / **60** 秒 |
| 波间 | 点击波波猫打开商店（游戏暂停） |
| 通关 | 第 10 波计时结束 → 胜利结算 |

### 难度与刷怪

敌人生命等数值受 **波次系数** 影响：1 → 2 → 4 → 8 → 12 → 24 → 30 → 36 → 44 → **50**（详见 [敌人](enemies.md)）。

- 怪物在屏幕外生成，向玩家移动  
- 生成间隔随场上敌人数动态调整（约 0.5～4 秒）  
- 击杀过慢时后续刷怪减少，过快则增加  

### 波次结束

计时结束后清理剩余敌人；场上金币 **30%**、经验宝石 **10%** 概率自动飞向玩家，其余消失。随后进入波波猫商店。

掉落详情见 [掉落物](items.md)。

### 剧情节点

| 波次 | 事件 | 效果 |
|:----:|------|------|
| 3 | 初遇卡莫 | 解锁卡莫选角；商店池加入激光枪、霰弹枪、激光剑 |
| 6 | 初遇阿女 | 解锁阿女选角；赠送花刀玉手与 **10 天赋点** |
| 10 | Boss 战 | [镰刀先生](enemies.md#镰刀先生) 加入战场 |

## 波波猫商店

每波开始前波波猫从左侧入场，**点击** 打开商店。

### 商品槽位（共 4 个）

| 槽位 | 规则 |
|------|------|
| 第 1 槽 | **必定** 为已解锁武器 |
| 第 2～4 槽 | 随机（见下表） |

| 商品 | 概率 | 效果 |
|------|:----:|------|
| 武器 | 70% | 从已解锁池随机 |
| 彩虹糖 | 14% | 立即 +1 天赋点 |
| 镜子 | 14% | 复制背包中已有武器 |
| 升级套件 | 2% | 指定武器 +1 级 |

价格随波次与等级浮动，并受角色 **魅力**（商店减免）影响。购买一律扣钱，无首购免费、无屏蔽武器。

### 刷新

| 来源 | 次数 |
|------|------|
| 开局 | 3 次 |
| 每次升级 | +3 次（可跨波存储） |
| [猫薄荷](items.md#植物) 掉落 | +1 次 |

点击「刷新」消耗 1 次机会并重 roll 四槽；用尽后按钮变灰。

---

## 升级与天赋

拾取 [经验宝石](items.md#经验宝石) 升级。每次升级：

- **+2 天赋点**（在 `M` 面板分配八维属性，单项上限 20）  
- **+3 商店刷新次数**  
- 生命恢复约 10% 最大生命  

属性说明见 [属性](attributes.md)。满 20 属性或特殊条件可解锁武器，见 [已实现武器 · 解锁列](weapons/implemented.md)。

## 胜负与存档

| 结果 | 行为 |
|------|------|
| 生命归零 | 失败 → 返回主菜单 |
| 完成第 10 波 | 通关 → 返回主菜单 |

武器解锁、角色解锁与剧情进度写入本地存档，跨局保留。武器解锁条件汇总在 [已实现武器](weapons/implemented.md) 表中。

## 与早期设计的差异

| 早期设计 | 当前版本 |
|----------|----------|
| 30 波 | **10 波** |
| 自选初始武器 / 装甲臂 | 与角色绑定 |
| 三名角色开局可选 | 仅阿猴，第 3 / 6 波解锁 |
| 商店首购免费、可屏蔽武器 | 已取消 |
| 完整元素反应 | 部分实装，见 [Buff 与伤害](buffs-and-damage.md) |
