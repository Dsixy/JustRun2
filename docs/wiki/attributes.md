# 属性

> 补丁：**以代码公式为准**。UI 中「感知」对应设计文档「眼界 / 眼光」；「熟练」对应「dexterity」（代码字段 `dexterrity`）。

## 能力属性（0～20）

通过 **天赋点** 在 `M` 面板分配；单项最高 20，满 20 可触发对应 **武器解锁**（见 [解锁](unlocks.md)）。

| 属性 | 作用 |
|------|------|
| **体力** | 最大生命值 |
| **力量** | 部分近战 / 远程武器伤害 |
| **启迪** | 法术与元素类武器伤害 |
| **敏捷** | 移动速度、暴击率 |
| **魅力** | 交互类效果、**商店降价** |
| **感知** | 金币 / 经验加成、拾取范围 |
| **坚韧** | 护甲（减伤） |
| **熟练** | 精准度、攻速、拾取范围 |

## 基础属性

| 属性 | 说明 |
|------|------|
| **等级** | 0 级起，上限 **30**（代码中经验获取在 29 级后停止） |
| **生命值** | 归零即失败 |
| **护甲** | 减伤公式：`实际伤害 = 原始伤害 × 100 / (100 + 护甲)` |
| **移动速度** | 像素/秒 |
| **金币加成** | 拾取金币额外倍率 |
| **经验加成** | 经验球额外倍率 |
| **暴击率** | 与武器基础暴击叠加 |
| **拾取范围** | 自动拾取判定半径 |

## 天赋点获取

| 来源 | 数量 |
|------|------|
| 升级 | **+2** / 级（补丁：代码为主） |
| 彩虹糖（商店） | +1 |
| 阿女首次剧情 | +10 |
| 作弊模式 | 大量（测试用） |

升级时在角色头顶飘字提示，**不自动打开** 天赋面板。
