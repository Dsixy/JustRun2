# 商店

波波猫（Popocat）在 **每波开始前** 从左侧入场；**点击** 后打开商店。游戏暂停直至关闭。

> 补丁决议见下表。

## 槽位

共 **4 个毛丛**（ShopSlot）：

| 槽位 | 规则 |
|------|------|
| **第 1 槽** | **必定** 为已解锁武器 |
| **第 2～4 槽** | 按权重随机（见下） |

## 商品权重（代码现状）

| 类型 | 概率 | 说明 |
|------|------|------|
| 武器 | **70%** | 从 `WeaponUnlock` 池随机 |
| 彩虹糖 | 14% | 立即 +1 天赋点 |
| 镜子 | 14% | 复制已有武器 |
| 升级套件 | 2% | 武器 +1 级 |

设计文档原为 77% / 10% / 10% / 3%，以 **代码** 为准直至改动。

## 定价

**补丁：以代码 `Utils.set_price()` 为准**（分段随机，随波次与玩家等级变化），**非** 文档公式 `(5+波次×等级)×rand(0.8,1.2)`。

实际价格 × **角色商店减免**（魅力相关）。

## 购买规则

| 规则 | 状态 |
|------|------|
| 一律扣钱 | ✅ 补丁确认（无首购免费） |
| 每波屏蔽一种武器 | 🚫 补丁删除 |
| 购买武器 → 解锁图鉴 / 商店池 | ✅ |

## 刷新

| 来源 | 次数 |
|------|------|
| 开局 | 3 次（可调整） |
| **每次升级** | **+3**（补丁：文档为主）⚠️ 待实现 |
| 猫薄荷（掉落） | +1 |
| 作弊模式 | 大量 |

点击「刷新」消耗 1 次刷新机会并重 roll 四槽；次数用尽后按钮变灰。

## 植物 · 猫薄荷

拾取 **猫薄荷** 增加 **1 次** 商店刷新（非「自动刷新当前商品」）。

## 旅游猫咪台词

首局提示：「点击左侧波波猫打开商店」（仅一次）。
