# 操作与界面

## 键盘

| 按键 | 功能 |
|------|------|
| `W A S D` | 移动 |
| 鼠标左键 | 攻击（多数武器朝鼠标方向） |
| `B` | 打开 **背包 + 武装板** |
| `M` | 打开 **天赋面板**（分配天赋点） |
| `T` | 打开 **图鉴** |
| `Esc` | 关闭当前 UI |

## 战斗 HUD

- **第 N 波** / **剩余时间**
- 生命、经验、等级、金钱
- 左下角快捷键提示

## 背包操作

1. 按 `B` 打开背包与武装板  
2. 将武器/道具 **拖拽** 至武装板空槽  
3. 同 ID、同等级的两把武器拖到一起可 **合成升级**（最高 4 级）  
4. 拖至删除区可丢弃  

## 商店

- 每波开始前，**波波猫** 从左侧出现  
- **点击波波猫** 打开商店（首局有一次操作提示）  
- 购买后物品进入背包；关闭商店后继续战斗  

## 升级反馈

升级时不再弹出天赋面板，而是在角色头顶 **飘字**：

- 「升级！」
- 「天赋点+2」

需要分配时自行按 `M`。

## 作弊模式

选角界面勾选 **「使用饱饱猫的力量」**（`GameInfo.cheat`）：

- 大量初始金钱 / 天赋点 / 商店刷新  
- 打开背包时左侧出现 **武器快捷栏**（点击即装入背包，用于测试）
