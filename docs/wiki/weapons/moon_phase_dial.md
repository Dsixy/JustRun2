## 设计文档（权威玩法 — 📋 未实装）

### 核心：月能循环

- 每次攻击积累 **1 点月能**，对 **10 取模**（0～9）。
- 根据月相释放不同攻击：

| 月能 | 月相 | 效果 |
|------|------|------|
| **0** | 新月 | 周围降下 **寒霜** |
| **1～4** | 上弦月 | **弧光**，伤害随月能变化 |
| **5** | 满月 | **月华**；对冰元素单位施加 **辉彩** |
| **6～9** | 下弦月 | 弧光（类似上弦） |

### 辉彩（设计）

- 承受者 **死亡后爆炸**；**50%** 额外掉落 **中级经验宝石**。

### 设计数值

| 项目 | 值 |
|------|-----|
| 暴率 / 暴伤 | 5% / 200% |
| 新月 | `5 + 3×等级 + 2×启迪` |
| 上/下弦 | `15 + 10×等级 + 3×启迪 + (2+0.5×等级)×月能` |
| 满月 | `25 + 20×等级 + 5×启迪` |
