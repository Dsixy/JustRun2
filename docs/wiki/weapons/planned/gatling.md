## 概述

**极高攻速**、**高负载** 贯穿武器。每 **50 发** 需 **重新装填**（基础装填时间 **3 s**）。携带者 **移速 -50%**。**不参与装甲臂攻击轮回**。

---

## DPS（设计）

| 阶段 | DPS |
|------|-----|
| 0 级 | **10** |
| 满级 | **70** |

---

## 基础数值

| 参数 | 值 |
|------|-----|
| 精准度 | **0.7**（不随熟练成长） |
| 攻击间隔 | **0.1 s** |
| 伤害 | **1 + 1×等级 + 0.2×熟练** |
| 子弹速度 | **800** |
| 弹匣 | **50 发** / 装填 **3 s** |
| 移速惩罚 | **-50%** |
| 暴击 | 5% / 150% |
| 伤害类型 | **物理伤害** |

---

## 升级（0 → 4 级）

| 等级 | 效果 |
|------|------|
| **1** | 子弹上限 **+20** → 70 发 |
| **3** | 子弹上限 **+50** → 120 发（相对 0 级累计 +70） |
| **4** | 玩家 **向攻击反方向移动** 时，移速 **+50%** |

---

## 实现缺口

无弹匣/装填/移速 debuff/反向移速检测/独立 `_process` 射击。
