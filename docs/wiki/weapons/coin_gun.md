# 金币枪

| 字段 | 值 |
|------|-----|
| **ID** | 19 |
| **资源 Key** | `coin_gun` |
| **中文名** | 金币枪 |
| **解锁** | 单局获得 6000 金币 |

> 你的意思是说，我撒出去的钱，全都会消失吗？
---

## 效果

发射一颗金币子弹。金币数 >50/100/200/500/1000 时，消耗 1/2/3/5/8 金币，额外发射 1/2/5/8/14 金币子弹。

- 等级1：金币消耗-1
- 等级2：基础子弹数量+2
- 等级3：金币子弹获得伤害加成，数值为0.1%*玩家当前金币数量
- 等级4：当玩家持有金币>2000时，攻击转为发射金币雨：消耗 100 金币，发射 5 波弹幕，每波弹幕有20+3个子弹
