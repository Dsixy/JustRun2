# 霰弹枪

| 字段 | 值 |
|------|-----|
| **ID** | 2 |
| **资源 Key** | `shotgun` |
| **中文名** | 霰弹枪 |
| **解锁** | 设计为初始拥有；代码未默认解锁 📋 |

---

## 效果

以扇面向鼠标方向发射多颗霰弹。

- 等级1：霰弹数量 +2（5 → 7）
- 等级2：暴击伤害增加 50%
- 等级3：霰弹数量 +3（7 → 10）
- 等级4：攻击变为 3 波齐射，攻击间隔降至 0.6s

---

## 数值

| 等级 | 基础伤害 |
|------|------------|
| 0 | 3 |
| 1 | 6 |
| 2 | 10 |
| 3 | 18 |
| 4 | 30 |

- 单颗伤害：基础伤害 + 0.6 × 等级 + 0.2 × 熟练
- 0 级单发弹数：5；4 级：10 颗 × 3 波
- 子弹速度：700
- 精准度：0.8（固定）
- 攻击间隔：0.9s（4 级 0.6s）
- 基础暴击率：5%
- 基础暴击伤害：150%（2 级后 200%）

---

## 代码 ↔ 文档差异

- 场景 L1 写「额外 1 颗」，代码 `bulletNum += 2`。
- 场景 L4 写「基础伤害 +2」，代码 **无固定 +2**，仅改波次与间隔；等级加成来自 `calculate_damage()` 公式。
- 扇形散布使用 `Vector2.from_angle(i - bulletNum/2)` 叠加方向，非纯角度扇形。
