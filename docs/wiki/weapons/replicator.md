# 复制器

| 字段 | 值 |
|------|-----|
| **ID** | 26 |
| **资源 Key** | `replicator` |
| **中文名** | 复制器 |
| **解锁** | 携带两个相同武器通关 |

---

## 效果

辅助型武器：轮到它攻击时，令装甲臂槽位中 **位于其后的下一把武器** 额外攻击；无法复制自身。

- 等级1：攻击间隔 −0.4s（2.0 → 1.6s）
- 等级2：复制攻击暴伤 +150%
- 等级3：复制攻击暴率 +100%；攻击间隔 −0.6s（→1.0s）
- 等级4：复制 2 次；攻击间隔 −0.4s（→0.6s）

---

## 数值

| 参数 | 值 |
|------|-----|
| 直接伤害 | 0 |
| 复制次数 | 1（4 级 2） |
| 攻击间隔 | 2.0s → 0.6s（见升级） |
| 额外暴伤 | 0 → 1.5（2 级） |
| 额外暴率 | 0 → 1.0（3 级） |
