## 设计文档（权威 — 📋 目标玩法）

### 概述

攫取身前敌人并 **向前方甩出**；对 **甩动路径上** 敌人造成伤害。偏 **百分比生命** 与 **力量** scaling 的控制/处决型武器。

### 解锁

解锁 **大力士** `strongwoman`：单局击杀 **3000** 📋 TODO。

### 设计数值

| 参数 | 值 |
|------|-----|
| 攻击间隔 | **1.5 s** |
| 基础暴击率 | **20%** |
| 暴击伤害 | **200% + 2.5×熟练%** |
| 伤害公式 | `5 + 敌人(10% + 5×等级%) 最大生命值 + 2×力量` |

### 升级（设计）

| 等级 | 效果 |
|------|------|
| 1 | 抓取范围 +100% |
| 2 | 投掷距离 +500 |
| 3 | 投掷伤害 +（2×熟练% 敌人最大生命） |
| 4 | 攫取范围内 **所有** 敌人，**随机** 向周围丢出 |
