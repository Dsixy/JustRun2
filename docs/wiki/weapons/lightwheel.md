# 极速光轮

| 字段 | 值 |
|------|-----|
| **ID** | 24 |
| **资源 Key** | `lightwheel` |
| **中文名** | 极速光轮 |
| **解锁** | 待补充 📋 |

---

## 效果

发射一个飞轮，飞轮飞行到最大距离后会留在原地进行切割，若场上存在其他切割状态的飞轮，会同样唤起并向光标方向飞行。

二次唤起会消耗一格电量，飞轮初始拥有2格电量。电量为零时，唤起会转而去收回飞轮

场上最多拥有两个飞轮。

- 等级1：场上飞轮最大数量+1
- 等级2：飞轮大小增加 60%
- 等级3：场上最大飞轮数量格外增加2个
- 等级4：场上最大飞轮数量再增加1个；飞行距离增加300；飞轮初始电量加2；飞轮切割时造成额外雷电伤害。
