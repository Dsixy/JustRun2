## 概览

激光枪是 **指向鼠标的瞬时激光束** 武器：在玩家节点下生成激光，`scale.y` 由 0 展开至 `laserScale` 再收回，对路径上敌人造成单次伤害。间隔随 **熟练** 缩短；高等级加宽光束、减 CD、**3 道** 扇形齐射。设计为 **火焰伤害**；代码未传 damage type，燃烧 Buff 代码被注释。

## 代码实现

### 武器脚本 `laser_gun.gd`

| 变量 | 说明 |
|------|------|
| `laserScale` | 初值 **0.4**（光束半宽 scale.y） |
| `laserNum` | 初值 1；L4 → 3 |
| `extra` | L2 设 **0.4**，参与间隔 |
| `baseCritDamage` | **1.5**（非设计 2.0） |

**`attackInterval`（getter）**

```text
1.2 - 0.02 * dexterrity - extra
```

**`upgrade()`**

| 等级 | 代码 |
|------|------|
| 1 | `laserScale += 0.4` → 0.8 |
| 2 | `extra = 0.4` |
| 3 | `laserScale += 1.0` → 1.8 |
| 4 | `laserNum += 2` → 3 |

**`attack()`**

- 目标 = 鼠标位置。
- 每道激光：`angle = baseAngle + 0.3*i - 0.3*laserNum + 0.3*int(laserNum/2) + 0.3`（3 道时扇形展开）。
- `laser.init(Vector2.from_angle(angle), damage, laserScale)`。
- `modulate = Color8(200, 240, 255)` 视觉偏冷色。

### 激光 `laser.gd`

- 初始 `scale = Vector2(20, 0)`，0.1s 展开 y 至 `laserScale`，0.3s 收回并销毁。
- 命中敌人 `be_hit(damage.copy())`。
- `burning_buff` 附加 **已注释**。

### 剧情 `scene.gd` → `carmor_come()`

- 对话结束后实例化激光枪，解锁并尝试放入 `player.inventory`。
