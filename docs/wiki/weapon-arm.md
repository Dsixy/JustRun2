# 装甲臂

装甲臂是角色的 **武器调度器**：从武装板读取已装备武器，按 **分组顺序** 轮射，组间有冷却间隔。

> 补丁：装甲臂与角色 **固定绑定**，开局不可更换。

## 机制

1. 武装板有 **4 个槽位**，玩家从背包拖入武器  
2. 装甲臂将槽位划分为若干 **攻击段**（`content`）  
3. 每段内武器 **同时或依次** 开火（视段内配置）  
4. 段与段之间等待 **装甲臂攻击间隔**（受 Buff 等影响）  

### 示例（阿猴 · 悟空臂）

设计文档描述的基本轮次：

```
霰弹枪 ×1 → 冷却 → 手枪 ×2 同时 → 冷却 → …
```

实际起手为 **手枪** 预装于第二段单槽。

## 各角色装甲臂

| 角色 | 资源 ID | 攻击段结构 | 特点 |
|------|---------|------------|------|
| 阿猴 | `awu_kong` | `[槽0]` → `[槽1]` → `[槽2]` → `[槽3]` 分段 | 经典三段式，中间段常装主武器 |
| 卡莫 | `mor` | `[槽0]` → `[槽1, 槽2, 槽3 同时]` | 第二段 **三枪齐射** |
| 阿女 | `lady` | `[槽0, 槽1]` → `[槽2, 槽3]` | 前后 **配对双段** |

## 与武器的关系

- 部分武器（如加特林设计）可 **脱离装甲臂轮次** 独立射击 — 📋 多数未实现武器概念  
- 武器 **攻击间隔** 参与段内最长等待计算  
- 拖回背包或合成升级后需重新 **`update_weapons()`** 刷新轮次  

## 操作

按 `B` 同时打开背包与武装板 → 拖拽武器到上方槽位 → 关闭后继续战斗。
